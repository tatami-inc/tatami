Detailed Description

template<bool ROW>
class tatami::Workspace< ROW >

Base workspace class.

Template Parameters
ROWWhether to perform row-wise extraction.

This provides a workspace for extraction of entire vectors from Matrix instances, either for each row (if ROW = true) or column (otherwise). Instances of this class cannot be constructed; it is only provided as a base class for Matrix::new_row_workspace() and Matrix::new_column_workspace() methods to return appropriate objects.
